# Логотипы клиентов

Сюда складываем логотипы клиентов для секции «Наши клиенты» на главной.

## Формат
- **PNG** с прозрачным фоном (рекомендуется) или **SVG**
- Размер: примерно **300×200 px** (или любая пропорция, картинка впишется в карточку с сохранением пропорций)
- Имя файла на английском, без пробелов: `vladsk.png`, `wb.svg`, `yandex.png`

## Как добавить нового клиента
1. Положить файл сюда: `src/assets/clients/имя-файла.png`
2. В файле `src/data/site.js` в секции `clientsSection.items` добавить запись:
   ```js
   {
     logo: new URL('@/assets/clients/имя-файла.png', import.meta.url).href,
     logoAlt: 'Название компании',
     title: 'Заголовок (необязательно)',
     description: 'Описание сотрудничества'
   }
   ```
